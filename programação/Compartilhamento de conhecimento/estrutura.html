Estruturação

Criei este artigo, visando melhorar o conhecimento sobre a estrutura básica do HTML5 preocupando com a semântica do conteúdo HTML pensando no SEO para buscar mais atratividade para os sites.

Levando-se em consideração a estruturação básica do HTML ele possui o <head> que é a estrutura "inicial" que fica por trás da página. É nela quem estão as característica de linguagem, viewport, entre outras. Porém, como ainda não tenho muito conhecimento nesta área ainda, vou pular direto para o <body>.

Estrutura da tag <body>

É nessa estrutura que tudo acontece!

A tag <body> é responsável por apresentar todo o conteúdo do site. É nela que vamos descorrer sobre o assunto, criar formulários, apresentar imágens, vídeos, enfim as possíbilidades são infinitas.

Porém, para capturarmos a atenção dos robôs de busca, como os do google ou bing, é aconselhavel que nosso HTML possua semântica (sentido). Antigamente os desenvolvedores de sites, criavam suas estruturas contendo tag's que na época não traziam consigo essa característica de semântica, o que transfomava o código às vezes em um ninho de amafagafos. Por isso, com a nova versão do HTML5, veio para auxiliar nessas estruturas e facilitar que os sites de busca, encontrem de forma mais eficaz informações dentro de cada site.

Estrutura semântica

A estrutura que sugiro que pratiquemos é esta:


                        <body> (Demonstrando o início do corpo do site.)
                        <header> (Cabeçalho)
                            <h1>Contendo Informações de cabeçalho</h1>
                            <nav>
                                <a href="#">Botões</a>
                                <a href="#">Botões</a>
                                <a href="#">Botões</a>
                            </nav>
                        </header>
                        <main> (Parte principal do site)
                            <section> (Seções)
                                <article> (Artigos)
                                    <h1>Título</h1>
                                    <p>Texto qualquer que faz parte do artigo.Texto qualquer que faz parte do artigo.Texto qualquer que faz parte do artigo.</p>
                                </article>
                            </section>
                        </main>
                        <footer> (Rodapé)
                            <p>Texto de rodapé<p>
                        </footer>
                        </body>
